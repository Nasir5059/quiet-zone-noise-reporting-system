# Quiet Zone Noise Reporting System (QZNRS)

This is a Software Engineering course project designed to report and manage noise complaints in university quiet zones such as libraries, hostels, and study halls.

## Features
- Anonymous noise complaint submission
- Admin login and dashboard
- Complaint status management
- Simple and user-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript
- Browser LocalStorage
